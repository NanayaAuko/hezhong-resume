專案與作品集

Personal Wealth & Financial AI System

個人理財與資產預測系統

這是一套結合網頁爬蟲、資料庫與 AI 運算技術，自主開發的個人化資產管理儀表板，具備現代化前後端架構，完美展示了「資料擷取 > 儲存運算 > AI 分析 > 視覺化」的全端開發能力。

利用 Python 與 C# 撰寫排程爬蟲，自動擷取台股 ETF (如 0056, 00878) 數據，透過類似 SAP DTW 邏輯自動拋轉至 SQL Server。

撰寫 Stored Procedures 處理高複雜度的資產配比與歷史損益運算。

以 .NET Core Web API 為核心後端，並整合 LM Studio API 進行市場數據 AI 輔助分析。

前端逐步導入 Blazor 框架，實現響應式與高互動性的數據視覺化儀表板。
