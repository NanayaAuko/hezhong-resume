Network Operations Center (NOC)

維運自動化解決方案 | Python & Scrapy & AI

系統核心優勢

針對大規模虛擬化環境 (VMware/OLVM) 進行主動式探測。結合 Scrapy 爬蟲技術，解決了監控數據散落在不同網頁管理介面的痛點，實現統一彙整。

  • ● Scrapy 採集: 自動化爬取伺服器狀態儀表板數據。
  • ● AI 故障分析: 整合 LM Studio 對異常 Log 進行語意分析與分類。
  • ● 預警推送: 偵測到異常指標時，即時透過 Webhook 發送警示訊息。

代碼展示：Python 監控核心邏輯

Python
import psutil, requests

# 定時採集核心硬體指標並發送 Webhook 預警
def monitor_and_alert():
    cpu_usage = psutil.cpu_percent(interval=1)
    if cpu_usage > 90:
        requests.post(WEBHOOK_URL, json={
            "text": f"⚠️ 警報：伺服器負載過高！目前 CPU：{cpu_usage}%"
        })

if __name__ == "__main__":
    monitor_and_alert()
