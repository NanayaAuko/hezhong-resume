專案與作品集

Intelligent Server Monitoring & AI Synthesis

智慧伺服器監控與 AI 數據整理系統

針對企業內部維運需求，開發一套自動化採集與 AI 分析工具。透過爬蟲技術解決監控數據雜亂、人工彙整不易的痛點。

利用 Python 撰寫自動化排程爬蟲，擷取多台伺服器網頁數據與系統日誌。

整合 LM Studio API (在地化 LLM)，對非結構化的狀態數據進行自動化彙整與關鍵異常提取。

設計類似 SAP DTW 的資料拋轉邏輯，將整理後的 AI 分析結果回填至 SQL Server 資料庫。

前端使用 Blazor 框架建置儀表板，提供即時的系統健康報告與維運建議。
