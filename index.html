結合網頁爬蟲、資料庫與 AI 運算技術，自主開發的個人化資產管理儀表板，具備現代化前後端分離架構。

技術實作細節

  • 爬蟲自動化拋轉：利用 Python 與 C# 撰寫排程爬蟲，自動擷取台股 ETF (如 0056, 00878) 數據，並透過類似 SAP DTW 邏輯自動拋轉至 SQL Server。
  • 高效能數據運算：撰寫 Stored Procedures 處理高複雜度的資產配比與歷史損益運算。
  • AI 輔助分析：以 .NET Core Web API 為核心後端，整合 LM Studio API 進行市場數據的 AI 輔助分析與重點摘要。
  • 現代化前端：前端逐步導入 Blazor 框架，實現響應式與高互動性的數據視覺化儀表板。
