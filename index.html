FeedFormulaForSQLite

畜牧飼料配方管理系統 (Open Source)

專案概述

這是一套專為畜牧從業者設計的單機版配方管理軟體。透過 **C# WinForm** 建置視覺化介面，並採用輕量化的 **SQLite** 儲存各類原料成分（如粗蛋白、纖維等）與配方歷史紀錄。

代碼展示：SQLite 資料庫讀取邏輯

C# / SQLite 
// 讀取原料營養成分資料
public DataTable GetIngredientData(string name) {
    using (var conn = new SQLiteConnection("Data Source=FeedDB.db")) {
        conn.Open();
        string sql = "SELECT * FROM Ingredients WHERE Name = @name";
        return conn.Query(sql, new { name = name }).ToDataTable();
    }
}
