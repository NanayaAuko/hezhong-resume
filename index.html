}

技術重點

這是一套專為畜牧業設計的現代化配方管理工具。採用 **C# WPF (Windows Presentation Foundation)** 開發，利用其強大的樣式自訂與數據綁定特性，並結合輕量化 **SQLite** 資料庫進行數據持久化。

// C# 使用 Dapper 調用 SQLite
public IEnumerable<Ingredient> GetAll() {
    using (var conn = new SQLiteConnection(ConnectionString)) {
        return conn.Query<Ingredient>("SELECT * FROM Ingredients");
    }
}
